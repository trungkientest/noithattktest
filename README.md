# SEO Starter Project (4 pages)

Mục tiêu: Cho sinh viên thực hành SEO kỹ thuật theo checklist:
- UI/UX cơ bản
- External CSS/JS (không inline)
- Meta tags + canonical + viewport
- Internal links + breadcrumb
- XML sitemap
- Schema (JSON-LD): BreadcrumbList, Product, FAQPage, Organization, LocalBusiness
- Mobile-friendly

## Cấu trúc
- index.html (Home)
- category.html (Category)
- product.html (Product list)
- product-detail.html (Product detail)
- assets/css/style.css
- assets/js/main.js
- assets/images/logo.svg
- assets/images/product-giay-sneaker-abc-den-1.svg
- sitemap.xml

## Hướng dẫn chạy
Chỉ cần mở `index.html` bằng trình duyệt.

Nếu muốn chạy local server (khuyến nghị để test đường dẫn):
- VS Code → Extensions → Live Server → "Go Live"

## Bài tập bắt buộc
1) Mỗi trang viết lại: title, meta description (~150–160 ký tự), keywords.
2) Thay `https://example.com/...` trong canonical/schema/sitemap bằng domain thật khi deploy.
3) Đảm bảo: 1 H1/trang; H2/H3 hợp lý; alt ảnh mô tả đúng.
4) Thêm tối thiểu 8 internal links (anchor text rõ ràng).
5) Test PageSpeed (Mobile + Desktop) và sửa ít nhất 1 đề xuất.
6) Test Schema bằng công cụ Structured Data của Google và chụp kết quả.
